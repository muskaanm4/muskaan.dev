# Digital Portfolio Muskaan.dev

A personal portfolio website built with plain HTML, CSS, and JavaScript.

## Features
- Responsive multi-page layout
- Interactive homepage and about page animations
- Work page with expandable project previews
- Contact form with social links and playful Easter egg interactions

## Project Structure
- `index.html` — homepage
- `about.html` — about page
- `work.html` — work/projects page
- `contact.html` — contact page
- `css/` — stylesheets
- `js/` — JavaScript interactions
- `img/` — images and assets

## Run locally
Open https://muskaanm4.github.io/Muskaan.dev/ in your browser to view this site. 

## Notes
This project is a static site, so no build step is required.
